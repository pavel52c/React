Задача: Реализовать страницу, на которой в виде списка в три стоблца показываются одинакового размера блоки. Блок содержит: наименование, которое занимает одну шестую часть по высоте от блока, описание - объемом в 2/3 от общего числа и нижняя часть - объемом в 1/6.
Все границы выделены линиями.

на странице должны быть следующие элементы управления:
 - добавление нового блока с информацией (все поля необязательны к заполнению)
 - удаление блока (можно в виде крестика на каждом блоке)
 - поиск на странице с выводом найденных на странице блоков.

Структура данных, которая находится в localStorage:
[
  {
    title: "H0EAD1",
    description: "Decription 1",
    created: "01.01.2021"
}, ...
]
